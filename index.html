About Myself

Passionate about storytelling and language, I've always been fascinated with breaking things apart and figuring out how they work. In recent years I've grown an interest in writing and the English language. Analyzing works and writing short stories has been a hobby of mine for some time. Outside of that, I've grown a massive interest in computers and programming.

Programming languages are just as interesting to me as spoken language and here you can find some of my experiments with both! This site is my exploration into HTML, CSS and Javascript in the future. In the past, I converted an old laptop into a CentOS webserver with a simple LAMP stack configured on it. This website is inspired by that one. Outside of that I'll detail my interests in Java, Python, and Webdev in general.

Ever curious and always humble, I seek to figure out where I can improve and strive to work towards my goals. If you have any input on my active projects or wish to talk to me about working together, please contact me below.

Knowledge

Python

Python is my biggest interest at the moment. It has expanded my knowledge of programming from just making basic, simple programs in Java to exploring ways to make programs interact with each other, automate simple tasks, or calculate useful things.

Java

My introduction to programming, Java is the language that got me started. Common OOP concepts like encapsulation and modularity interested me a ton. The language interests me because OOP concepts seem to work best in a team environment. Additionally, I have a fairl easy time visualizing the development process of a program so the division of labor in a large project seems fairly easy in Java.

Webdev

I've started doing challenges on FreeCodeCamp which has been immensely helpful in learning CSS, HTML and Javascript in the future. Combining my interests in design, programming and usability, I've found front-end web development to be a lot of fun.

Linux/Git

Setting up a LAMP stack on an old laptop of mine turned out to be an absolute blast. While not being my primary goal in learning programming, I used the opportunity to familiarize myself with Linux and Git.
